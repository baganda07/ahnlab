AhnLab MDS

  • AhnLab MDS

高级的恶意软件响应解决方案
收集-分析-监控-响应一体化

  • 产品介绍
  • 主要功能
  • 产品规格

AhnLab MDS 是检测如高级持续威胁APT(Advanced Persistent Threat) 日益进化的最新安全威胁，并通过专用Agent实时响应的下一代安全解决方案。

   
  

主要特征



      
恶意软件及异常流量检查系统
AhnLab MDS(Analyzer)
  • - 收集和分析主要互联网服务协议文件 (HTTP, SMTP, SMB, FTP 等)
  • - 监控入站和出站流量
  • - 使用多引擎恶意软件分析提高了检测可信度
  • - 通过基于虚拟机的分析，分析新种恶意软件 ( 同时运行最多10 个虚拟机)
  • - 装载了 Non-PE 型恶意软件分析引擎 (MS Office, pdf reader 等)
  • - 检测并阻止通过受感染计算机或C&C 通信访问有害网站
  • - 检测受感染计算机的DDoS 流量
  • - 支持多种的网络接口选项 (1G copper/fiber 及 10G fiber)
综合监控及日志管理系统
AhnLab MDS Manager(Data Viewer)
  • - 通过仪表板提供安全状态及主要事件信息
  • - 实时确认恶意软件和异常流量状态
  • - 对于受感染的文件，发送修复命令( 自动/ 手动)
  • - 对于被恶意软件感染的主机，发送提取可疑文件命令
  • - 显示事件日志、IP 地址、行为细节 ( 文件/ 进程/ 注册表/ 网络)
  • - 下载检测到的文件
  • - 对VM 分析过程及C&C 检测内容，提供基于PCAP 的数据包捕获，并下载PCAP文件
  • - 与第三方安全管理系统 (SIEM, ESM) 互操作的系统日志转发
修复及Agent管理系统
AhnLab MDS Manager(Host controller)
  • - 通过Controller，提供 Agent 功能及补丁更新
  • - 对 Agent 数量增加可以弹性响应的服务器群形式的扩张功能
  • - 对未安装 MDS Agent 的主机自动发布安装文件
  • - 根据Controller 分析结果传送命令
  • - 对大量Agent 进行有效的负载均衡管理
专用 Agent
AhnLab MDS Agent
  • - 修复受感染的主机上的恶意软件 (自动/ 手动)
  • - 实时确认恶意软件和异常流量状态
  • - 对于恶意软件分析中的PE 文件，提供‘运行保留 (execution holding)’功能
  • - 分析通过加密网络流量 (如SSL) 流入或通过U 盘和内部网络流入到端点的新种恶意软件
  • - 恢复已删除的文件
  • - 提供对个别/ 所有主机的公告
  • - 安装Agent 后无需重启系统
