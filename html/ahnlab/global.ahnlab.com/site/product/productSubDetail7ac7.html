Overview

AhnLab TrusGuard integrates firewall, IPS, VPN, anti-virus, and anti-spam security features with a unique self-defense system against DDoS attacks. It is designed for efficient and secure connectivity with internal systems through an IPSec/SSL VPN. AhnLab offers a full lineup of TrusGuard to protect any business environment, including TrusGuard 22000 for full-scale, customizable corporate network security.

 

Features

AhnLab TrusGuard Features Details
Feature Details
Firewall
  • - Stateful Packet Inspection methods
  • - Blacklist and Whitelist filters
  • - Guarantees constant, independent performance regardless of the number of policies or sessions
  • - Supports multiple NAT features; Static/Dynamic NAT, Excluded NAT, NAT Traversal
  • - Separate QoS policy can be applied by IP/Port
  • - Intuitive configuration and easy management
  • - Schedule-based policy configuration; One time, Daily, Weekly, Monthly, Yearly and Specific Period
  • - Guarantee of availability: Active-Active, Active-Standby HA (with/without L4 switch)
  • - Password-based authentication, internal one-time password, and external RADIUS/LDAP/AD linkage authentication
  • - Supports VoIP dynamic port (H.323/ SIP)
  • - Policy validity check
  • - Supports Session Shaping
  • - Supports User-ID based firewall policy settings/management
IPS
  • - Packet flow-based anomaly detection/block against network attacks and malicious codes
  • - Signature-based intrusion prevention (maintenance of more than 6,000 signatures)
  • - Signatures updated daily
  • - Behavior-based intrusion prevention
  • - Supported by ASEC, AhnLab Security Emergency response Center
  • - User-defined rules and signatures
  • - Customized IPS policy can be applied by network zones
  • - Bot prevention feature
  • - Zero-day attack protection, via a 3-step defense system
Application Control
  • - More than 1000 application signatures supported
  • - Access and login control on the main application users
  • - Supports 250 specialized applications in Korea
C&C Detect and Block
  • - Detects and blocks C&C server access
  • - Supports more than 100,000 cloud-based C&C blacklist signatures
  • - C&C blacklist daily updated
Anti-virus
  • - Detects and blocks file-based viruses and malicious codes (via the embedded AhnLab V3 engine)
  • - Detects and blocks file-based viruses, Trojans, worms, spyware, adware, phishing attempts, spam, malicious sites, and other Internet threats
  • - Prevents email viruses at the gateway (outbreak prevention)
  • - Supports major protocols; HTTP, SMTP, POP3, and FTP
  • - Scans compressed files (maximum of 5 compressions) and specific file extensions
  • - Supported by around the clock monitoring and analysis at ASEC
  • - 24/7 update via the Contents Delivery Network (CDN)
Anti-Spam/
Web Filtering
  • - Blocks Spam Mail, supports SMTP and POP3
  • - Detects and blocks spam with a global anti-spam engine and Real-time Blacklist(RBL)
  • - Blocks spam based on user-defined keywords (includes wildcard and regex support)
  • - Supports whitelist of Sender IPs and E-mail addresses
  • - Synced with DB of harmful websites information and website filtering
  • - Synced with DB of websites information which distribute malicious codes
DDoS Protection
  • - Dedicated defense engine for DDoS attacks
  • - Prevents TCP, UDP, and ICMP flooding
  • - Prevents HTTP vulnerability attacks
  • - Prevents various types of DDoS attacks
IPSec VPN
  • - Supports manual key, IKE, and IKEv2
  • - Includes encryption algorithms for 3DES, AES (128, 192, 256), SEED, and ARIA
  • - Includes authentication algorithms for SHA 1, SHA 2 (256, 384, 512), and HAS 160
  • - Uses a hub and spoke, star, or mesh topology
  • - Includes NAT Traversal support, dead peer detection, and prevents Replay attack
  • - Includes firewall and IPS interlocking
SSL VPN
  • - Gateway-to-Client VPN
  • - Includes firewall and IPS interlocking
NAC Network access control (NAC) in conjunction with the V3 antivirus solution :
  • - Access control for computers that have not installed APC Agent and provides automatic redirection to the installation page
  • - Network quarantine for malware-infected computers and forced remediation via APC Agent
  • - Malicious traffic blocking as well as real-time, automatic V3 updates
IPv6
  • - IPv4/IPv6 dual stack support
  • - IPv4/IPv6-enabled simultaneous processing
  • - IPv6 networking, routing, and packet filtering support
  • - IPv6 static and dynamic Routing (RIPv1/v2, OSPF)
  • - IPv6 translation
  • - IPv6 stateful inspection-based packet filtering
  • - IPv6 log collection and analysis
Monitoring
  • - Real-time graphical monitoring and analysis of network, firewall, IPS, antivirus, antispam, and related log data
  • - AhnLab TrusGuard Manager and TrusAnalyzer (installation of optional S/W required)

Advantages

Scales up to protect high-performance networks

Capable of handling growing network traffic, thanks to optimization for high-performance multicore environments

Ensures network stability

Protects network resources (such as, websites, database servers, applications servers, and client machines) from unknown network attacks with the 3-step defense.

Covers IPv6 network environments

Complete support for IPv6 networks

Reduces total cost of operation (TCO)

Offers cost cuts compared to integrating multiple security products Relieves operational and labor costs associated with managing multiple security solutions

Increases productivity and network efficiency

Eliminates unwanted traffic via access control and restriction of programs and sites that are hazardous or not related to work Optimizes network traffic to lower network operation costs

Centralizes responses to network threats

Routes VPN traffic through the head office for enhanced security Centralizes management of quarantines and treatment of infected computers via the AnhLab Policy Center (APC)

Enhances protection for branch offices

Includes IPS, antivirus, and anti-malsite features that are distributed to branch offices for enhanced protection against attacks

Proactive Defense

TrusGuard provides a total security with an exclusive response system of ACCESS (AhnLab Cloud Computing E-Security System). With the real time threats analysis process, proactive defense is ensured on the unknown and Zero-day attacks.
