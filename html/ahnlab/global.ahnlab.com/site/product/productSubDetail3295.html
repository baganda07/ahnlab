Overview

AhnLab Online Security (AOS) is designed exclusively to provide security for online transactions. Regardless of the security protocols used on a local computer, AOS provides complete security for the online transaction. It is a proven web transaction security solution of 15 years which monitors and responds quickly against the numerous hackers who attempt illegal financial transactions and steal financial information.

Unlike a typical single-purpose security solution, AOS combines three key security elements into an innovative security solution that prevents the leakage of information and thwarts attempts to compromise online financial transactions.

AOS is comprised of 4 security services which include Anti-keylogger, Firewall, Anti-virus/spyware and Secure Browser. The services ensure that advanced security features are in place to safeguard users’ financial and personal information.
