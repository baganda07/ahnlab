Symptom and Summary

Trojan/Win32.CryptoWall.C1317179 encrypts user files on system such as images and documents, and demands ransom for decrypting the files via anonymous network and payments by BitCoin. 

Detail Information

* Method of Infection

This malware does not self-propagate. It is likely that the system could be infected when a user downloads an executable file from a spam mail. Another common vector of infection is drive-by download which is exploited by visiting web pages with vulnerabilities.

 

* Symptoms after Execution

[Creating Files and Execution]

This malware duplicates itself the following path: 

- %AppData%\[RandomName].exe

It then injects its code for execution to the following system processes:

- explorer.exe 

- svchost.exe

 

[Adding a Windows Registry Entry]

It adds the following value on Windows registry to be executed whenever Windows starts.

- KEY: HKCU\Software\Microsoft\Windows\CurrentVersion\Run\[RandomName] 

- DATA: %AppData%\[RandomName].exe

 

[Removal of Volume Shadow Copies]

Volume Shadow Copies are backup copies of files or volumes on a Windows system, created either manually or automatically. This malware removes all Volume Shadow Copies on the system by executing commands with vssadmin.exe (a Windows’ system utility that manages Volume Shadow Copies), preventing the restoration of the files that have been encrypted by itself.

 

[Network Connections]

This malware acquires a key from the following remote server for the encryption of the files:

- 66.71.1**.*

Note) Addresses have been partially omitted with *s.

 

[Encryption and Other System Changes]

This malware encrypts various types of user files such as document files and image files. Examples of the file extensions that are encrypted by this malware are:

.hwp, .doc, .docx, .xls, .xlsx, .ppt, .pptx, .pdf, .jpg, .psd, .txt, .zip

This malware also changes the name of the files after the encryption to the following format:

[RandomName].[RandomExtension]

Additional files that contain guides for paying ransom are created on Desktop of the system, including:

- HELP_YOUR_FILES.PNG 

- HELP_YOUR_FILES.TXT 

- HELP_YOUR_FILES.HTML
