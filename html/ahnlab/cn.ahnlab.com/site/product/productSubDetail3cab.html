Overview

AhnLab HackShield for Online Game 2.0是一款世界领先水平的网络游戏防外挂解决方案，AhnLab 20多年的网络安全技术确保了这一产品的全球顶尖品质。AhnLab HackShield for Online Game 2.0可以阻止各类外挂程序对游戏的入侵和破坏，保障游戏公平稳定的运行，确保游戏的长久生命力。

  • - 通过阻止外挂程序对游戏的影响，确保玩家可以长期的享受游戏的乐趣。
  • - 确保稳定而公平的游戏环境，增强玩家对游戏及其运营商的信心。
  • - 实现对新型外挂及其他安全威胁的迅速、及时响应，确保客户游戏业务的稳定运行。
  • - 阻止由Autoplay和Bot等类型的恶意程序而导致的销售收入的降低。
  • - 通过防外挂安全管理操作最简化的实现，提高游戏管理的效率。
  • - 通过拦截导致服务器负荷增加的恶意程序，减少服务器购置成本。
  • - 为客户建立一个无须担心外挂影响，可以专注于新游戏开发和全球业务的新环境。

产品特征

根据特征码感知外挂

基于 AhnLab 防病毒技术，利用引擎检测是否有外挂拦截内存入侵 拦截在内核级别上直接跟踪内存、操作结果值等方式的非法入侵

拦截加速外挂

为了拦截加速外挂，随时确认 Microsoft 进程层面上的系统时间和操作系统上的逻辑时间之间的差距。当发生一定时间以上的差距时，通知游戏客户端。

防止反编译

为了防止利用 Debugger 分析游戏程序、非法入侵的试图行为，拦截所有 Debugger 的跟踪。

拦截 Message Hooking

拦截使用 Windows 的 Message Hooking 功能，获取非法入侵所需的事件。

防止文件的篡改

对 HackShield 进行初始化或正在执行游戏时，确认 HackShield 本身的文件是不是起初分发的文件。

拦截 Auto Mouse

拦截 Auto Mouse，防止自动鼠标输入，操作游戏程序，减少游戏服务器的过多负荷。

导入 Packer

导入 Packer 对 HackShield 的文件以及相关DLL文件进行加密和压缩，防止修改 HackShield 的代码。

提供智能升级

随时提供拦截或检测非法入侵的引擎。游戏客户端通过指定的更新服务器，可下载拦截或检测非法入侵的引擎。
