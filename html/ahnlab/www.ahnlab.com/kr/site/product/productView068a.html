제품소개

모든 파일, 데이터베이스, 애플리케이션을 위한 데이터 암호화

Vormetric Encryption은 조직 내외의 다양한 보안 위협으로부터 중요 정보를 보호하는 동시에 개인정보보호와 관련된 컴플라이언스 이슈에 대응할 수 있도록 하는 데이터 암호화 솔루션입니다. 이를 위해 엔터프라이즈 서버 상의 민감한 데이터를 암호화하고, 정보에 대한 접근제어를 실행하고, 보호된 데이터에 누가 액세스하고 있는지에 대해 보고서를 작성할 수 있도록 도와줍니다. Vormetric Encryption은 리눅스, 유닉스, 윈도우 등 주요 플랫폼들을 모두 지원하며 물리적, 가상화, 클라우드 환경 어디서나 사용이 가능합니다. 또한 단일 솔루션을 통해 정형 데이터와 비정형 데이터 모두를 통합된 키 관리로 보호할 수 있습니다.

AhnLab은 Vormetric과의 파트너십을 통해 보다 강력하고 효율적인 데이터 암호화 솔루션을 제공합니다.

주요기능

Vormetric Encryption 구성요소

Vormetric Encryption은 Vormetric Data Security Manager 어플라이언스와 고도의 보안이 요구되는 데이터에 액세스하는 서버 또는 엔드포인트 상에서 사용되는 Vormetric Encryption Agent 소프트웨어로 구성됩니다.

Vormetric Data Security Manager, Vormetric Encryption Agent의 주요기능 상세내역
구성요소 Vormetric Data Security Manager Vormetric Encryption Agent
주요기능
  • - 하드웨어 타입 (HSM)
  • - 암호화 모듈 중앙 관리 콘솔
  • - 중앙회된 정책 설정 및 관리
  • - 암호화 키 관리 & FIPS
  • - 암호화 대상에 대한 감사Vormetric Encryption은 물론Vormetric Key Management, VormetricVault를 위한 하나의 콘솔 기능 제공
  • - 소프트웨어 타입 (OS 소스 기반)
  • - 민감한 데이터 보호를 위한 정책 적용
  • - OS 소스 기반의 암호화 모듈
  • - File & Raw File에 대한 암호화
  • - 암호화 대상에 대한 Access 제어 (시스템 상의 접근제어)
  • - 암호화 대상 서버(호스트) 보호
  • - Linux, Unix, Windows 등 주요 OS 지원

Vormetric Encryption 주요기능

  • - 소프트웨어 최적화 및 Intel® AES-NI & SPARC Niagara Crypto 등의 하드웨어 암호 가속화 기술을 지원하여 제로에 가까운 성능 오버헤드 실현
  • - 분산된 엔터프라이즈 전반의 파일 시스템, 데이터베이스, 애플리케이션에 대한 중앙 정책 관리 및 감사(Audit)
  • - 애플리케이션 및 데이터베이스 트랜스패런트(Transparent) 데이터 보안 – 애플리케이션, 데이터베이스, 스토리지 변경이 필요 없음
  • - 플랫폼, 애플리케이션, 기기 전반에 대한 강력한 암호화, 접근제어, 키 관리
  • - 물리적, 가상화, 클라우드 환경을 비롯한 분산된 환경 전반을 위한 유연한 보안 인프라스트럭처
  • - 정교한 접근제어, 감사, 호스트 무결성 기능을 통해 컨피규레이션이 자유로운 보안 및 정책 실행
  • - Oracle 및 Microsoft SQL Server에 대한 TDE(Transparent Data Encryption) 암호화 키 관리와 기타 암호화 키들의 키볼팅(Key Vaulting)을 제공하는 Vormetric Key Management가 결합된 Vormetric Data Security 인프라스트럭처

특징/장점

Vormetric Encryption의 중앙화된 정책 및 키 관리는 조직이 하나의 공통된 데이터 보안 절차와 일관된 보안 태세를 확보하도록 함으로써 다양한 시스템 상의 여러 데이터 유형을 보호하는데 따른 관리 업무를 최소화시켜 줍니다. 또한 고성능 Vormetric Encryption 기술을 기반으로 한 가벼운 아키텍처를 통해 단 5일 만에 구축되어 파일과 데이터베이스에 대한 암호화, 접근제어, 키 관리, 감사(Audit), 호스트 무결성(Host Integrity) 기능을 모두 제공합니다.

모든 유형의 데이터 보호

Vormetric Encryption은 정형(structured) 데이터베이스 정보는 물론 PDF, 스프레드시트, 스크립트, 이미지, 오디오/비디오 레코딩, ETL(extract-transform-load) 데이터와 같은 비정형(unstructured) 파일을 비롯한 모든 유형의 데이터를 보호합니다. Vormetric의 파일 레벨(file-level) 암호화 방식은 애플리케이션과 데이터베이스에 트랜스패런트하므로, 애플리케이션 변경 및 스토리지 조정에 따른 비용과 부담을 피할 수 있습니다.

강력한 보호

Vormetric Encryption은 검증된 암호화 및 키 관리의 결합을 통해 권한을 가진 사용자(privileged users)를 관리하고, 이들이 저마다 필요로 하는 데이터에 액세스할 수 있도록 보장합니다. 동시에, 필요하지 않은 데이터는 해당 사용자에게 보이지 않게끔 합니다. 이를 통해 엔터프라이즈는 자사의 데이터에 방화벽을 세우고, 승인된 사용자와 애플리케이션만 데이터에 액세스할 수 있도록 보장할 수 있습니다. 승인되지 않은 사용자와 애플리케이션의 액세스는 차단됩니다. Vormetric의 강력한 키 관리는 SOD(separation of duties)를 제공함으로써 조직이 PCI DSS, HIPAA/HITECH, 기타 정보보호 관련 법안들을 비롯한 엄격한 규제 요건을 준수할 수 있도록 해줍니다.

효율성

기존의 암호화(Encryption) 방식은 많은 프로세싱 리소스를 소비하고, 애플리케이션 성능을 저하시킵니다. Vormetric은 소프트웨어 튜닝(software tuning), 그리고 Intel® AES-NI, SPARC Niagara Crypto 모듈을 비롯한 하드웨어 암호 가속화 (hardware cryptographic acceleration)를 통해 암호화에 따른 성능 저하를 거의 '제로'에 가깝게 만들었습니다. 암호화 성능 오버헤드를 획기적으로 줄임으로써 엔터프라이즈는 민감한 데이터를 보호하는 동시에 애플리케이션 성능과 SLA (service level agreements)를 유지할 수 있습니다.

간편성

Vormetric Encryption은 구축, 관리, 운영이 간편합니다. Vormetric Encryption은 중앙화된 정책 및 키 관리를 통한 최소한의 운영 오버헤드만으로 물리적 서버, 가상 서버, 클라우드 서버를 포함한 다양하고 분산된 구축 환경을 광범위하게 지원합니다.
