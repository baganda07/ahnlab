V3 Mobile Plus

안전한 스마트폰 뱅킹, 금융 거래 등을 위한 모바일 보안 솔루션

V3 Mobile Plus

  • 브로슈어 보기
  • 제품소개
  • 주요기능
  • 특징 / 장점
  • 사용환경
  • 제품연혁

제품소개

  • - AhnLab V3 Mobile Plus는 스마트폰을 통한 다양한 거래 행위시 이와 관련된 정보 유출, 불법 과금 등을 유발하는 스마트폰 악성코드에 대한 실시간 탐지를 제공하는 모바일 트랜잭션 보안 솔루션입니다.
  • - 언론 보도에 따르면 스마트폰 가입자 수는 2010년 연말을 기준으로 600만 명을 돌파할 것으로 예상됩니다. 이와 더불어 스마트폰을 이용한 온라인 쇼핑 등은 물론 계좌 조회 및 계좌이체, 증권 거래 등 스마트폰 금융 거래 또한 크게 급증할 것으로 예측됩니다.
  • - 스마트폰을 이용한 전자금융거래의 경우 고객의 개인 정보, 계좌 정보 등이 입력, 전송되기 때문에 안전한 이용 환경 구축이 반드시 필요합니다.
  • - AhnLab V3 Mobile Plus는 금융 및 공공기관 애플리케이션, 스마트폰 기반의 모바일 오피스 프로그램과의 간편한 연동으로 안전한 스마트폰 트랜잭션 환경을 제공합니다.
  • - 금융감독원의 ‘스마트폰 전자금융서비스 안전대책’ 준수를 지원하기 위해 안드로이드 플랫폼 기반 스마트폰의 루팅 검진 모듈(Rooting Certification Module)’을 별도 지원합니다.
  • - 또한 스마트폰 기반의 모바일 오피스 프로그램과의 간편한 연동으로 안전한 모바일 오피스 환경 구축에 기여합니다.
  • - 대한민국 보안 브랜드 1위 안랩의 20년간 축적된 안티바이러스 DB와 국내 최초 모바일 보안 솔루션의 10년 노하우를 통해 효과적으로 모바일 악성 코드를 분석, 대응합니다. 
  • - AhnLab V3 Mobile Plus는 Android, Windows Mobile 등 다양한 운영체제를 지원합니다.

주요기능

간편한 애플리케이션 연동

  • - 금융 및 공공기관 애플리케이션, 스마트폰 기반 모바일 오피스 프로그램과의 간편한 연동(SDK ) 을 통해 안전한 모바일 환경 제공

악성코드 무력화

  • - 24시간 365일, 긴급대응팀(ASEC)비상 체제 운영
  • - 실행 시 자동 업데이트 및 실시간 검사
  • - 신/변종 악성코드 발견 시 즉각적인 업데이트 실시

금융감독원 지침 준수 지원

  • - 안드로이드폰 루팅 검진 모듈 별도 지원

사용자 편의성 제공

  • - 데이터 네트워크, Wi-Fi 등 무선망으로 엔진 업데이트
  • - 전자금융거래 애플리케이션과 함께 실행 및 종료

특징/장점

AhnLab - AhnLab V3 Mobile Plus 특징/장점

AhnLab V3 Mobile Plus 동작방식

  • 사용자편의성 + 악성코드 실시간 탐지 :

    실시간 모바일 악성코드 탐지, 사용자 편의성 제공

    스마트폰은 3G, Wi-Fi 등 유무선에 항상 연결되어 있기 때문에 악성코드 전파 경로로 이용되기 쉽습니다. V3 Mobile Plus는 애플리케이션과 동시에 수행/종료되어 이용이 편리하며, 실시간으로 악성코드를 탐지하여 개인정보 유출을 방지합니다. 특히 스마트폰을 이용한 전자금융거래의 경우 고객의 개인 정보, 계좌 정보 등이 입력, 전송되기 때문에 안전한 이용 환경 구축이 반드시필요합니다.

  • 금융 감동원 지침 준수 지원 :

    금감원 요구사항 준수를 위한 ‘루팅 검진 모듈’ 별도 지원

    금융감독원은 스마트폰을 이용한 전자금융거래와 관련해 안드로이드 플랫폼 기반 스파트폰의 루팅(Rooting)를 통해 발생할 수 있는 보안 사고에 대응하기 위한 대책 수립을 요구하고 있습니다. 이와 관련해 V3 Mobile Plus는 안드로이드 폰의 루팅 검진 모듈(Android Rooting Certification Module)을 별도 지원하고 있습니다.

  • 모바일 오피스 보안 :

    안전한 모바일 오피스 환경 구현

    스마트폰을 통한 기업 네트워크로의 악성코드 전파가 심각한 사회적 위협으로 대두되고 있어 모바일 오피스 구현에 걸림돌이 되고 있습니다. V3 Mobile Plus는 금융 및 공공기관 애플리케이션은 물론, 모바일 오피스 프로그램과 간편하게 연동되어 편리하고 안전한 스마트폰 이용 환경을 제공합니다.

  • 대 고객 이미지 향상 :

    20년간 축적된 DB, 국내 ‘최초’ 모바일 백신의 10년 노하우

    안랩 20년간 축적된 안티바이러스 DB와 국내 최조의 상용 모바일 보안 솔루션으로서 10년간 쌓아온 노하우는 신뢰성 있는 모바일 환경 구축을 통한 경쟁력 강화에 기여합니다. 대한민국 보안 브랜드 1위의 안랩과 함께 대 고객 이미지를 향상에 기여합니다.
