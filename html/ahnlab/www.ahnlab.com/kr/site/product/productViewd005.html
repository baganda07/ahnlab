제품소개

AhnLab V3 MSS는 다차원 분석 플랫폼이 적용된
중소기업용 통합 PC 보안 솔루션입니다.
중소기업에 최적화된 클라우드 기반의 보안 솔루션으로,
별도의 보안 관리자가 없이도 쉽고 편리하게
사내 PC의 보안 현황을 파악, 대응할 수 있는 솔루션입니다.

주요기능

AhnLab V3 MSS는 V3 MSS Desktop(클라이언트)과 V3 MSS Security Center(관리서비스)로 구성돼 있습니다. 다차원 분석 플랫폼 기반의 V3 MSS Desktop으로 정확한 악성코드를 탐지 및 치료하며 PC 관리가 가능합니다. 웹 기반의 V3 MSS Security Center를 통해 사내 PC의 전체 사용 현황 및 보안 상태를 모니터링할 수 있습니다.

특징/장점

AhnLab - AhnLab V3 MSS 특징/장점

AhnLab V3 MSS는 관리의 효과는 증대시키면서도 비용을 최소화하는 장점이 있습니다.

다차원 분석 플랫폼 적용으로 한층 뛰어난 진단율을 자랑하며 사전 방역까지 가능합니다.

클라이언트 1copy의 가격으로 안티바이러스 및 안티멀웨어, 해킹 차단 등 다양한 보안 위협에 대응할 수 있으며 클라이언트 PC의 보안 상태까지 관리 가능한 특징을 지니고 있습니다.

  • 보안 관리 비용 최소화:사내 보안 관리자 없이 관리/도입 비용 최소화
  • 보안 관리 효과 증대:긴급한 보안 위협 발생시 신속 대응 가능
  • PC 보안부터 관리까지 중소기업 보안에 최적화:개별 PC 보호부터 중앙 정책 관리까지

무료체험

정품과 동일한 성능을 30일 동안 무료로 체험해 보세요.

  • - 평가판은 아이디당 1회만 신청할 수 있고, 30일 동안 사용 가능합니다.
  • - 사용기간이 만료된 후에는 평가판을 재설치 할 수 없습니다.
  • - 만료 후에도 제품을 계속 이용하는 경우 불법 소프트웨어 사용으로 간주되어 민 · 형사상의 처벌을 받을 수 있으니, 평가판 사용기간이 만료된 후에는 정품을 구매하여 사용하시기 바랍니다.

무료체험 안내

  • 1. [평가판 다운로드] 버튼을 클릭하면 V3 MSS 홈페이지로 이동합니다.
    [V3 MSS 무료체험] 페이지에서 로그인 후 [Trial 신청]을 클릭합니다.
  • 2. V3 MSS 홈페이지 내 [Security Center]에 접속하여 화면 안내에 따라 관리자의 정보와 설치 비밀번호를 설정합니다.
  • 3. [배포관리 > 설치 파일 배포]의 '파일 주소'를 설치할 사용자에게 메일로 발송합니다.
  • 4. 메일로 받은 '파일 주소'를 클릭하고, <사용자 인증> 화면에는 사용자 정보와 설치 비밀번호를 입력합니다.
  • 5. 사용자 인증이 완료되면, 설치파일이 다운로드 되고 설치가 완료되면 서비스를 이용하실 수 있습니다.
  • 6. V3 MSS 평가판은 최대 5유저(PC)까지 설치할 수 있습니다.
