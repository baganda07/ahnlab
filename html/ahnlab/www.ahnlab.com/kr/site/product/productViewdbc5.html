HP TippingPoint

제품소개

최신 네트워크 인프라 공격에 대한
강력하고 선제적인 방어 솔루션

BYOD, 클라우드 컴퓨팅, 애플리케이션 증가 등 IT 환경변화와 정교하고 지능화된 보안 위협의 증가에 따라 오늘날의 보안 관리자는 더욱 많은 위험을 관리하고 더욱 많은 공격 지점을 방어해야 하는 상황에 놓여 있습니다. HP TippingPoint는 2001년부터 높은 네트워크 성능 및 가용성을 보장하면서 선제적 네트워크 보안을 제공하는 in-line IPS 솔루션 개발에 집중해왔습니다. 2009년 이후 지속적인 차세대 침입방지 시스템(NGIPS) 개발을 통해 최대 20Gbps의 in-line, real-time 침입방지 기능을 제공하며 고객의 네트워크와 데이터 센터의 확장에 유연하게 대응하는 NGIPS의 새로운 기준을 제시하고 있습니다.

AhnLab은 HP와의 보안 솔루션 파트너십을 바탕으로 차세대 네트워크 침입방지 솔루션 제공하여 갈수록 진화하는 보안 위협에 대한 선제적이고 능동적인 대응 방안을 제시합니다.

<주요특징>

  • 1. 평판 디지털 백신(RepDV), 사용자 정의 IP, DNS 평판 엔트리, 위치 기반 정책 설정 기능
  • 2. SMS(IPS 관리 서버)를 통한 통합 운영 및 관리 편의성 제공
  • 3. 심층 패킷 검사(Deep packet inspection)을 통한 애플리케이션 인지, 가시성 확보, 통제 강화
  • 4. 애플리케이션 디지털 백신(AppDV), 웹 애플리케이션 디지털 백신(WebAppDV), 사용자 개발 필터 적용
  • 5. 특정 파일 타입을 검사하고 중요 정보를 보호하기 위한 콘텐츠 인지 및 통제

주요기능

1. TippingPoint N Platform NGIPS

보안 기능 및 성능 향상

  • - 100% 하드웨어 보안 플랫폼 ASIC, FPGA, Security Network Processor
  • - Control Plane과 Data Plane을 위한 별도의 하드웨어 탑재를 통한 네트워크 고가용성 지원
  • - 하드웨어 기반 IPv4, IPv6 보안 프로세싱
  • - Hitless OS Rebooting/Reload

향상된 설치 옵션

  • - 최대 11개 세그먼트까지 지원
  • - 10GE, 1GE copper, fiber, ZPHA, DC power options
  • - VLAN Translation을 통한 내부망 보안 지원

가상화 환경에서 보안 기능 제공

  • - 내부 가상 머신간의 유해트래픽 검출 (vController)

2. TippingPoint NX Platform NGIPS

보안 기능 및 성능 향상

  • - 2U 기반의 모듈형 포트 고밀도/고성능 10Gbps IPS
  • - 100Gbps IPS 성능 제공을 위한 2U 기반 Base 하드웨어 플랫폼(IPS Stacking)
  • - 차세대 ASIC/FPGA/Network Security Processor 탑재
  • - Control Plane과 Data Plane 분리, 네트워크 고가용성 지원
  • - 하드웨어 기반 IPv4, IPv6 보안 프로세싱, Hitless OS Rebooting/Reload
  • - X-Forwarded-For, Akamai True-Client-IP, Sflow 정보 연동 기능 제공

향상된 설치 옵션

  • - 4개 Type의 모듈형 IO 지원
  • - 10GbE 최대 16개, 40GbE 최대 4개 세그먼트까지 지원
  • - 40GbE 10GE, 1GE copper, fiber, ZPHA, DC power options
  • - (option) Built-In ZPHA IO 모듈 제공
  • - 네트워크 변경에 따른 I/O 모듈 선택 지원을 통한 유연한 설치와 Hot-swap 지원
  • - VLAN Translation을 통한 내부망 보안 지원

New Security Intelligence

  • - 컨텍스트 기반 보안 인텔리전스(평판, 위치기반 DV) 지원
  • - 웹 보안 스캐너와 연동을 통한 자동 웹 보안 필터 생성

특징/장점

업계를 선도하는 보안 연구소와의 협업: DVLabs와 Zero Day Initiative(ZDI)

DVLabs는 첨단 엔지니어링 및 분석을 통한 취약점 발견에 특화된 HP의 보안 연구소입니다. Zero Day Initiative(ZDI)는 발견된 보안 취약점을 책임 있게 공개하는 전세계 연구자들에게 보상을 제공 하기 위해 DVLabs가 운영하는 프로그램입니다. DVLabs는 발견된 보안 취약점과 연관된 모든 소프트웨어 벤더에게 발견 내용을 전달하는 동시에 NGIPS 필터를 제작하여 Zero-day 공격으로부터 고객을 보호합니다.

높은 신뢰성과 가용성

HP TippingPoint NGIPS는 최고 수준의 가용성을 유지하도록 설계되었습니다. 네트워크 장애 또는 내부 장치 오류 시에도 높은 네트워크 트래픽 속도를 보장합니다.

데이터 센터 및 코어 네트워크를 위한 고성능, 저지연 시스템

HHP TippingPoint NGIPS는 데이터 센터 및 네트워크 코어 보호를 위해 설계되었습니다. 이러한 미션 크리티컬 네트워크 영역의 경우, HP의 NGIPS 플랫폼은 40μs 이하의 지연시간으로 최대 20Gbps의 자동화된 인라인 검사를 수행할 수 있습니다. 즉, 성능 저하 없이 네트워크 장치, 가상화 소프트웨어, 운영 체제 및 애플리케이션을 사이버 공격으로부터 보호합니다.

높은 필터 정확성으로 낮은 오탐율 보장

HP TippingPoint NGIPS는 필터 정확성을 보장하기 위해 No False Positives와 No False Negatives 라는 두 가지 간단한 필터 작성 규칙을 사용합니다. 이러한 이유로 HP의 DVLabs 보안연구소는 알려진 악성코드를 포함한 전체 취약점을 보호하기 위한 필터 개발에 집중하고 있습니다. HP 취약점 필터는 탁월한 정확성을 바탕으로 합법적인 트래픽을 차단하지 않으면서 소프트웨어 취약점에 대한 모든 공격을 차단하여 네트워크를 보호합니다.

응급 패치 없이 Zero-day 공격으로부터 시스템 보호

HP 취약점 필터는 애드혹(ad-hoc)이나 응급 패치의 필요성을 제거해줍니다. 소프트웨어 취약점을 보호함으로써 고비용인 동시에 업무중단을 발생시키는 응급 패치 대신 정기적으로 예약된 프로세스에 의해 소프트웨어 패치를 구현할 수 있습니다. 최신 연구에 의하면, 날로 증가하는 취약점 수로 인해 클라이언트 측 애플리케이션에 패치를 지속적으로 적용하기 어려운 것으로 나타났습니다. HP의 NGIPS 플랫폼은 패치되지 않은 시스템 및 네트워크에 대한 취약점 보호를 통해 악성코드전이와 확산을 방지합니다. 또한 실제 환경에 보안 패치를 설치하기 전에 이를 테스트할 수 있습니다.

네트워크 대역폭 관리를 통한 성능 향상

HP TippingPoint NGIPS에는 네트워크 내 P2P나 미디어 스트리밍과 같은 비업무 애플리케이션 사용을 중지시키는 대역폭 관리 기능이 있습니다. 네트워크에서 악성 또는 비업무 트래픽을 지속적으로 제어함으로써 미션 크리티컬 애플리케이션에대한 네트워크 성능을 가속화합니다. 실제로 비업무 애플리케이션에 대한 rate-shaping을 통해 일부 경우에 40~70%까지 대역폭 가용성을 증가시킬 수 있습니다.

보안관리시스템(SMS)을 통한 통합 운영관리 기능 제공

다수의 NGIPS 장비를 통합 관리하기 위해 설치되는 보안관리시스템(SMS)을 통해 IPS 장비 관리(등록/수정/삭제), 보안 정책 통합 관리( Segment 및 group 별 관리), IPS 보안 이벤트 저장 및 모니터링 등 NGIPS 에서 발생하는 모든 행위를 저장하고 백업 및 복원할 수 있습니다.

손쉬운 설치로 시간과 자원 절약

HP NGIPS 플랫폼을 통해 네트워크 유지보수에 투입되는 시간과 자원을 절약할 수 있습니다. NGIPS와 보안관리시스템 (SMS)은 일반적으로 30분에서 2시간 내에 쉽게 설치할 수 있습니다. NGIPS는 네트워크 투명성을 위해 설계되었으며 IP 주소 또는 MAC 주소 없이 네트워크에 무결절 설치가 가능합니다.

단순하지만 강력한 보안 정책

HP NGIPS를 사용하면 보안 관리자가 정교하고 세밀하게 보안 정책을 관리할 수 있습니다. 관리자는 네트워크 세그먼트, VLAN, CIDR(Classless Inter-Domain Routing) 별로 특정 네트워크 보안 정책을 설정할 수 있습니다. 또한 고객은 NGIPS 플랫폼의 평판 기능 및 평판 디지털 백신(RepDV)을 활용하여 IP 주소와 DNS 명을 보안 정책 관리에 통합할 수 있습니다.

강력한 보안 리포팅 기능 제공

NGIPS 및 SMS에서의 리포트를 사용하여 관리자는 내부 및 외부 감사인에게 네트워크를 최신 위협에서 보호하는 방법을 보여줄 수 있습니다. 고객은 규정 및 내부 컴플라이언스 준수 요구사항뿐만 아니라 고객 네트워크에 대한 보다 강화된 보안을 확보할 수 있습니다.
