제품소개

엔드포인트 통합 관리 솔루션 AhnLab V3 Endpoint Security 9.0은
매체제어 기능을 제공함으로써 PC의 다양한 경로를 통해 유입되는
악성코드를 원천 차단합니다.



  • USB
    • USB 저장 장치 사용
    • USB 저장 장치 접근 로깅
  • 드라이브
    • 디스크 드라이브
    • 모뎀
    • 플로피 디스크 드라이브
    • CD/DVD 드라이브
    • 스마트 카드 리더
  • 네트워크
    • 유선 네트워크
    • 무선 네트워크
    • 외장 네트워크 어댑터

  •  

    • IEEE 1394(FireWine)
    • PCMCIA 어뎁터
    • 적외선 장치
    • Bluetooth 장치
    • COM/LPT 포트

또한 다차원 분석 플랫폼이 적용돼 있어 다양한 보안 위협으로부터 기업의 클라이언트 PC를 안전하게 보호하며, 사전 방역 기능을 제공해 안전한 컴퓨팅 환경 구축에 기여합니다.

  • - 매체제어 기능 제공, PC의 다양한 장치로 유입되는 악성코드 차단(APC/APC Appliance 연동 시)
  • - 다차원 분석 플랫폼 기반의 탁월한 진단율, 사전 방역 효과
  • - 스마트 스캔 기술로 신속 · 정확한 검사
  • - 초경량 엔진으로 PC 메모리 사용량 최소화
  • - 쉬운 컬러, 메인 화면에서 문제 한번에 해결 

주요기능

AhnLab V3 Endpoint Security 9.0은 매체제어를 비롯해 웹 보안, Active Defense 등 강력하고 다양한 보안 기능을 제공합니다.

매체제어

  • - 블루투스, USB 저장장치, CD/DVD 등 사용 시 제어 가능
  • - APC/APC Appliance 연동 기능(APC 최신 버전 시 사용 가능)

악성코드 진단

  • - 클라우드 기반의 ASD DB로 정확한 진단 및 실시간 치료
  • - 바이러스, 웜 등 악성코드 실시간 진단

네트워크 보안

  • - URL/IP 등 7억여 개 DB를 활용해 접속한 웹사이트에 대한 정보 제공
  • - 행위 기반 침입 차단, 네트워크 침임 차단, 유해 사이트 차단

행위/평판 진단

  • - 평판 정보를 활용해 안정성이 검증되지 않은 프로그램 실행 차단
  • - 비정상적인 패킷의 흐름을 모니터링, 이상 여부 판단

Active Defense

  • - 의심 프로세스, 프로그램 주요 행위 등 행위/평판 정보 제공
  • - 프로세스 및 파일에 대한 상세 리포트로 보안 가시성 확보

PC 최적화

  • - 메인 화면에 사용자가 손쉽게 운영하기 적합한 기능 구성 
  • - 보안 상태 주의/위험시 원클릭 해결, 완전삭제 기능

특징/장점

다차원 분석 플랫폼 기반의 AhnLab V3 Endpoint Security 9.0은 빠르고 정확한 악성코드 분석은 물론, 아직 알려지지 않은 신 · 변종 악성코드까지 진단합니다.

1. URL/IP 기반 탐지

악성 웹/피싱 사이트 방문 차단

악성코드가 PC로 다운로드 되기 전 시그니처 업데이트 없이 실시간 반영

2. 클라우드 기반 탐지

7억여 개의 DB 정보에서 실시간 확인

시그니처 업데이트 없이 실시간 반영

3. 시그니처 기반 탐지

DNA Scan 기술로 다양한 변종 진단

최초 발견 파일에 대해서만 사전 진단

4. 평판 기반 탐지

안정성 미확인 프로그램 차단

시그니처 없이 사전 방역, 평판 조건 사용자 선택

5. 행위 기반 탐지

100여 개의 악의적 행위 탐지

제로데이 취약점 원천 차단, 정확한 탐지

6. 엑티브 디펜스(Active Dfense)

실시간 분석 정보 제공

프로그램 활동 내역, 클라우드 자동 분석

무료체험

정품과 동일한 성능을 30일 동안 무료로 체험해 보세요.

  • - 평가판은 아이디당 1회만 신청할 수 있고, 30일 동안 사용 가능합니다.
  • - 사용기간이 만료된 후에는 평가판을 재설치 할 수 없습니다.
  • - 만료 후에도 제품을 계속 이용하는 경우 불법 소프트웨어 사용으로 간주되어 민 · 형사상의 처벌을 받을 수 있으니, 평가판 사용기간이 만료된 후에는 정품을 구매하여 사용하시기 바랍니다.

무료체험 안내

  • 1. 아래 [평가판 다운로드] 버튼을 클릭하여 설치파일을 원하는 폴더에 저장합니다.
  • 2. 저장한 파일을 실행하여 설치합니다.
  • 3. 설치 과정 중 <사용자 정보> 화면에는 사용자 이름과 회사명을 입력하고, 제품번호는 입력하지 않아도 됩니다.
    <사용자 인증> 화면에는 안랩 홈페이지의 아이디와 비밀번호를 입력합니다.
  • 4. 사용자 인증 완료 후 정품과 동일한 성능을 무료로 체험할 수 있습니다.
