Simplified Management of Business Endpoint Security

  • Overview
  • Features
  • Specifications

AhnLab Policy Center (APC) is a unified central management solution for managing our award-winning security applications for desktops and servers. From one location, easily manage and administer your endpoint security products to ensure seamless protection across the organization. APC provides flexible and scalable network configurations to fit any organization or network topology.



 

Endpoint Protection

Security Policy Management

APC provides total virus and malware protection, by collaborating with all AhnLab V3 products in your network. With a fully-integrated console, administrators can easily configure and deploy security protection profiles for managed domains or groups.

Proactive Defense

APC detects system security vulnerabilities in advance, including unknown security threats, by analyzing the following behaviors: (1) vulnerable shared folders or user accounts detected, (2) infected agent computer detected, or (3) an agent computer that violates the internet access control policies of AhnLab TrusGuard.

Central Management

Administrative Console

The single-point console lets administrators easily assess the security status of computers and servers across an entire corporate network. Administrators can configure settings, deploy APC agents, and control agent computers remotely.

Desktop Management

This feature configures settings for endpoint security program patches, Windows updates, and power management for agent computers to keep all the computers in your network up-to-date and to reduce energy consumption. Remote control support enables quick technical support and troubleshooting of networked computers.

Domain and Group Management

APC allows administrators to configure a domain or group based on internal organizational structures or other criteria. Simplified management of domains and groups provides effective control for large-scale networks and customizable security policies for each unit.

Monitoring & Reporting

Dashboard

The dashboard provides useful, at-a-glance information about your security status, including agent computer details, vulnerabilities, virus and spyware data, policy deployment, and scheduled tasks. Administrators can send notices, run engine updates, remotely control agent computers, distribute package files, run virus scans, or restart agents directly from the dashboard.

Reports

APC provides details about agent computers, including software and hardware configurations, vulnerable accounts and shared folders, security programs, and detected threats. Administrator can create custom reports with need-based queries and schedule reports at specified times.
