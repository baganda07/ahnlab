Smart Security for Android Applications and Markets

  • Overview
  • Features

AhnLab Mobile Smart Defense (AMSD) verifies the security of Android apps. AMSD uses both static and dynamic analysis to scan app permissions and filter out apps that exhibit malicious behaviors. It adds value for app markets and developers, as well as peace of mind for end users, by providing full control of app security on mobile devices.

  • - Identifies and prevents distribution of malicious apps
  • - Provides complete control of app security on mobile devices
  • - Easy to operate and manage app market


      

AhnLab Mobile Smart Defense verifies Android apps’ permissions, features, and developers as well for safe Android environment.

Verification of App Permissions

AMSD scans five different types of permissions to determine if application is asking for excessive permissions or if it has malicious purposes.



      

Verification of Malicious Features

Malicious features are identified by analyzing the API and evaluating the extent of the app’s permissions.

Verification of App Developers

Evaluations of app developers provide reputation data that is used to verify future apps by the same developer.
