Does an anti-virus program completely repair Code Red?

Hit
396

You need to end Explorer.exe (worm file) process running on your system and to repair Code Red. You must use an anti-virus program for Code Red, which ends the process first before repair.. After repair, you need to apply update patches for Index Server of IIS. For more details on Code Red, click here.
