Overview

アンラボのAndroidセキュリティを体験するなら、まずは10日間の無料体験版からいかがでしょうか。

＞V3 Mobile 2.0は次のURLからダウンロードできます。

ダウンロードする : https://play.google.com/store/apps/details?id=com.ahnlab.v3mobileforalj

​

近年スマートフォン、タブレットPCなどの利用者が急増し、デバイス環境はプラットフォームやアプリケーションを中心に大きく変化しています。これによりモバイル環境をめぐるセキュリティ上の脅威が増加しています。

壁紙の変更、ビデオプレーヤー、人気ゲームなど多数のユーザーが使用するアプリになりすましたマルウェアをはじめ、最近ではモバイルバンキングを狙うマルウェアも登場しました。これらのマルウェアはデバイスからユーザーの重要な情報を外部に流出したり、不正課金など金銭的被害を発生させ、モバイルセキュリティの重要性がさらに高まっています。​​

 

 

紛失・盗難・危険なWi-Fiアクセスポイントからも防御!

AhnLab V3 Mobile 2.0はスマートフォン、タブレットPCなどAndroid専用モバイルセキュリティソリューションアンチウイルス、着信/SMS遮断、盗難/紛失対策などモバイル Android に特化した製品です。

デバイスインストール型セキュリティ「AhnLab V3 Mobile 2.0」は、全世界で2000万台(＊)の導入実績です。

＊V3 Mobile、V3 Mobile Plusのユーザー総計（全世界）

主要機能

AhnLab V3 Mobile 2.0は、モバイルセキュリティソリューションとして最適化された機能を持っています。
特に、危険なWi-Fiの盗聴にもきちんと対応できるのはアンラボのV3 Mobile 2.0のみです。
