Overview

AhnLab V3 Mobile 3.0は、Android端末専用セキュリティソリューションです。

  • - アンラボは、スマートデバイスに関連して発生しうる脅威はもちろん、ユーザーのニーズやスマートデバイスメーカーのニーズまで多角的に分析し、Android端末専用セキュリティソリューションである「AhnLab V3 Mobile 3.0」を提供しています。
  • - AhnLab V3 Mobile 3.0は、Androidユーザーのニーズを積極的に反映しました。
  • - AhnLab V3 Mobile 3.0は、リモートロック(Anti-Theft)およびプライバシー保護機能など、スマートデバイスに最適化された機能が搭載された Android端末専用セキュリティソリューションであり、安全かつ便利なモバイル環境づくりに貢献しています。
  • - アンラボは、2001年よりモバイルセキュリティソリューションを開発/提供しており、その関連サービスも提供しています。
  • - AhnLab V3 Mobile 3.0は、現在韓国のメーカーが韓国で発売しているAndroid端末の約88％に対して標準搭載されています。

主要機能

OSバージョン：Android 4.0以降(但し、電話機能がないプレーヤーおよびタブレットPCでは一部の機能が正常に動作しない可能性があります。)

AhnLab V3 Mobile 3.0は、アンチウイルス機能およびアプリロック機能、アプリ権限チェック機能、個人情報クリーナー機能を10日間無料で使用できます。

1.ウイルススキャン

アンチウイルス

  • - リアルタイムスキャン(アプリケーションインストールの前後スキャン)
  • - クイックスキャン: インストールされたアプリケーションのスキャン
  • - 詳細スキャン: ファイルへのアクセス権限が許可された全体ファイルおよびPUAスキャン
  • - 自動およびスケジュールアップデート

ワンタッチセキュリティチェック

  • - エンジンアップデート
  • - マルウェアスキャン
  • - 端末の脆弱性(Root化の有無、提供元不明のアプリのインストール許可の有無、画面ロック使用の有無)スキャン
  • - アプリ権限チェック(インストールしたアプリケーションの権限チェック)

マルウェアダウンロードの検知

- APKファイルダウンロード時、マルウェアの有無を検知

2. 電話遮断

  • - 非通知電話の遮断
  • - 国際電話の遮断
  • - 特定番号の遮断
  • - 電話番号の先頭の2桁以上一致時、遮断
  • - 知らない電話番号から着信があった際、遮断の有無を選択可能

3. アプリロック

プライバシー保護のためのアプリケーションロック(パスワード)

4. 個人情報クリーナー

Webブラウザのアクセス履歴、クリップボード、キャッシュデータの削除

5. ギャラリー​管理

ギャラリー内の写真と動画ファイルの非表示

6. アプリ権限チェック

  • - デバイス管理者権限
  • - 盗聴防止(マイク、イヤホン、電話の受信/発信、録音機能が使用できるアプリをチェック)
  • - アプリ内課金
  • - 端末位置情報
  • - 連絡先へのアクセスなど
  • - インストールされたアプリケーションの15項目の権限ガイド

7. リモートロック

  • - SMSベースの紛失モードに切り替え
  • - アラート機能
  • - リモートロック、端末の所在位置を確認
  • - リモートによるデータ削除、リモートによる初期化
  • - SIMカードの変更時のアラート
  • - コマンド送信機能

製品の特徴

AhnLab V3 Mobile 3.0は、Android端末に最適化されたモバイルセキュリティソリューションです。

ウイルススキャン

· アンチウイルス

Androidプラットフォームを攻撃するマルウェアおよびユーザーの同意を得ずに個人情報にアクセスし流出する有害な応用プログラムを検知・削除します。

ユーザーの選択によるクイックスキャン/詳細スキャンおよび新しい攻撃手法が発見されるたびにエンジンをアップデートすることでユーザーのスマートフォンを安全に保護します。

· ワンタッチセキュリティチェック

エンジンアップデートと共に、マルウェアスキャンおよび端末脆弱性(Root化の有無、提供元不明のアプリのインストール許可の有無、画面ロック使用の有無)、インストールしたアプリケーションの権限チェックを一度に実行/確認することが可能です。

· マルウェアダウンロードの検知

ファイルのダウンロード時、マルウェアの有無を検知して、危険な場合は通知します。

電話遮断

非通知電話、国際電話、特定の番号、電話番号の先頭の2桁以上一致時遮断することが可能です。また、知らない番号から着信があった場合、遮断の有無を選択できます。

アプリロック

アプリケーションを選択し、個別にパスワード設定によるロックをすることで、プライバシーを保護することができます。

個人情報クリーナー

Webブラウザ履歴、クリップボード、キャッシュデータを削除することができます。

ギャラリー管理

ギャラリー内の写真と動画ファイルを非表示にすることができます。

アプリ権限チェック

デバイス管理者権限、盗聴可能なアプリ、アプリ内課金、端末位置情報、連絡先へのアクセスなど、インストール済みアプリケーションが持つ権限をガイドします。

リモートロック

端末紛失時にリモートコマンドを通じて、アラート、端末ロック、端末位置の確認、データの削除、初期化、SIMカード変更時の通知などができます。(データの削除、初期化、SIMカード変更時の通知の場合は、あらかじめ登録しておいた信頼できる番号を利用して制御および確認することができます。)

※ 注意 : ユーザーが実行したコマンドにより削除された端末のデータは復旧できません。また、アンラボは、ユーザーが設定したパスワードを別途管理していないため、[リモートロック]で設定したパスワードを紛失しないように大切に保管してください。

韓国初のモバイルセキュリティ提供ノウハウおよび専門性

  • - 韓国初の常用モバイルセキュリティソリューションの提供を通じて蓄積された専門性
  • - 約20年間V3アンチウイルスの提供を通じて蓄積された分析および対応システムが適用された専門性
  • - 特許技術である特定コード診断方法を利用して素早いスキャン速度を実現された専門性
  • - ビヘイビアベースにより潜在的なマルウェアに対応(Androidプラットフォーム)
